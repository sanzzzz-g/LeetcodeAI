# Code of Conduct

## 🌟 Our Promise

We’re building LeetcodeAI as a space where everyone feels welcome, included, and respected. Whether you’re new here or a seasoned contributor, your voice matters.

This repo is part of GSSoC 2026—a place where people learn, team up, try new things, and contribute to open source, no matter their background or experience.

---

## 🤝 What We Expect

Here’s what good behavior looks like:

- Show respect and support toward other contributors
- Lend a hand to beginners and anyone new to open source
- Give honest, helpful feedback during code reviews
- Write clean, clear, and well-documented code
- Stay professional in discussions and pull requests
- Listen to, and consider, different opinions and experiences

---

## 🚫 What’s Not Okay

We won’t stand for:

- Harassment, discrimination, or hateful talk
- Personal attacks or insults
- Spamming, trolling, or any disruptive conduct
- Plagiarism or copying work without giving credit
- Sharing private or sensitive info without consent
- Any toxic behavior in issues, discussions, or pull requests

---

## 💻 How to Contribute

To keep things running smoothly, we ask that you:

- Follow the existing structure and stick to our coding standards
- Test your code before opening a pull request
- Keep your pull requests focused and purposeful
- Write clear, straightforward commit messages
- Respect maintainers’ decisions and feedback

---

## 📌 Where This Applies

This Code of Conduct covers:

- GitHub issues
- Pull requests
- Discussions
- All project communication channels
- Any GSSoC contributions linked to this repo

---

## ⚖️ What Maintainers Do

Maintainers will:

- Review contributions fairly
- Keep the environment healthy and welcoming
- Act quickly on unacceptable behavior
- Help and guide contributors when possible

Sometimes, maintainers might remove comments, decline pull requests, or restrict access if someone breaks these rules.

---

## 📣 How to Report Issues

If you see or experience something that crosses the line, let the maintainers know—through GitHub issues, discussions, or direct messages.

We’ll treat every report seriously and respond with respect.

---

## ❤️ Why We’re Here

Our goal is to build an open, beginner-friendly, and collaborative AI-powered coding platform where everyone can learn, share, and grow—during GSSoC 2026 and long after.

Happy Contributing 🚀!
